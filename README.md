# PandasDataFrames-Project-Repository


## 🐍 Retail & Economic Data Analysis with Python (Pandas Project)

## 📌 Overview
This project was completed as part of a **Data Technician Bootcamp** and focuses on analysing data using Python, particularly with the **Pandas library**.

The project explores and analyses two datasets:
- 📊 **Student CSV dataset**
- 🌍 **GDP (Nominal) per Capita dataset**

Using these datasets, the project demonstrates how Python can be used to **clean, manipulate, and visualise data**, uncovering patterns and insights relevant to retail, demographics, and economic indicators.

---

## 🛠️ Key Skills Demonstrated

### 📚 Data Handling with Pandas
- Loaded and explored datasets using `pandas`
- Inspected data structure using:
  - `.head()`, `.info()`, `.describe()`
- Handled structured data in DataFrames for analysis

---

### 🔍 Filtering & Subsetting Data
- Filtered data based on conditions (e.g. values, categories)
- Used:
  - `.loc[]` → label-based selection  
  - `.iloc[]` → position-based selection  
- Extracted subsets of data for focused analysis (e.g. specific countries, student groups)

---

### 🧹 Data Cleaning & Preparation
- Managed missing data using:
  - `dropna()` – removing incomplete rows  
  - `fillna()` – imputing missing values  
- Ensured datasets were clean and usable for analysis  
- Standardised and prepared data for consistent processing  

---

### 🔄 Data Manipulation & Analysis
Applied core Pandas methods to transform and analyse data:
- `groupby()` – aggregated data by categories (e.g. country, student attributes)  
- `sort_values()` – ranked data by key metrics  
- Created calculated columns for deeper insights  
- Compared trends across datasets (e.g. economic vs demographic indicators)

---


## 📈 Use Case & Analysis
